Alternating groups

The table below summarises the currently recorded block-transitive designs associated with this family of groups. Each row gives group-level classification counts and, whenever a computational data file is available, links directly to the raw GAP file.

Click anywhere on a data row to open the raw GAP file. Use Download .g to start a direct browser download.
The entry in the Name column is the GAP list containing the designs stored in that file.
For an explanation of the information stored in each data file, see How to read.
The References column is currently being developed. Relevant sources will be added and updated progressively as the bibliography and database records are reviewed.
The references provided are not intended to be exhaustive, and the inclusion of a reference does not necessarily indicate the first publication in which the corresponding design was constructed.
In the enumeration columns, — means that the number of designs with the corresponding property has not been computed or is not available from the data file.
The designs for the groups indicated by * are obtained under some restricted conditions.
