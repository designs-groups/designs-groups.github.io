Software and Packages

The computations underlying this database were carried out with GAP and make substantial use of the DESIGN and GRAPE packages. The project also draws on methods from design theory, finite group theory, permutation group theory, and computational algebra.

We gratefully acknowledge the developers and maintainers of GAP and its packages, whose work makes this database possible.

Software resources

  • GAP — Groups, Algorithms, and Programming
    A system for computational discrete algebra, with particular emphasis on computational group theory.
    GAP website
  • DESIGN — The Design Package for GAP
    A GAP package for constructing, classifying, partitioning, and studying block designs.
    DESIGN package page
  • GRAPE — GRaph Algorithms using PErmutation groups
    A GAP package for computing with graphs and groups, including structures arising from groups, finite geometries, and designs.
    GRAPE package page
